# LinAim

LinAim is a native Linux Aim Trainer built from scratch using C++ and OpenGL.

Inspired by popular aim trainers like Aim Lab and competitive shooters like Valorant, this project aims to provide a lightweight, high-performance training environment specifically optimized for Linux users.

## Tech Stack
* **Language:** C++ (Standard 20)
* **Graphics API:** OpenGL 3.3+
* **Window Management:** GLFW
* **Mathematics:** GLM
* **Build System:** CMake
